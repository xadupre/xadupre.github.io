yet-another-onnx-builder — torch exporter coverage

These reports compare how the yet-another-onnx-builder (yobx) torch exporter behaves against the alternative torch→ONNX exporters (dynamo-ir, tracing, new-tracing, …) on a set of model cases, aten operators and PyTorch functions.

Reports bundled with the documentation

The following reports are rebuilt with every documentation build and reflect the latest state of the project shipped with this site.

case_coverage

Side-by-side exportability of test model cases across the available torch→ONNX exporters (yobx, dynamo-ir, tracing, …).

open report

op_coverage

Per-operator coverage of the ONNX operators produced by the torch exporters when converting the test model cases.

open report

supported_aten_functions

List of the torch.ops.aten.* functions supported by the yobx torch interpreter.

open report
