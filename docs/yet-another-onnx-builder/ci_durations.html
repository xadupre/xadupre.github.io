CI Job Durations (Past Two Months)#

This page shows the duration of CI workflow runs over the past two months for the yet-another-onnx-builder repository. Durations are fetched from the GitHub REST API and plotted as time-series charts — one chart per CI workflow.

Note

This page queries the public GitHub REST API. No authentication token is required for a public repository, but anonymous requests are subject to rate-limiting (60 requests/hour per IP). When the API cannot be reached (offline build, rate-limit exceeded, …) the chart will be empty and a warning is printed to the console. Retrieved runs are cached per workflow as CSV files for two weeks in the user cache directory.
