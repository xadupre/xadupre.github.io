class Greater#
#include <include_logical_kernels.h>

Element-wise Greater comparison with multidirectional broadcasting. Inputs may be FLOAT, INT8, INT16, UINT8, UINT16, UINT32 or UINT64 (both inputs must share the same dtype); the output is BOOL (one byte per element, 0 or 1). Mirrors the upstream ONNX Greater reference implementation (np.greater).
