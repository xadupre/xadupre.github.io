proto_utils.h
=============

General-purpose protobuf utility templates covering debug string formatting
(:cpp:func:`onnx::ProtoDebugString`), byte-buffer deserialization
(:cpp:func:`onnx::ParseProtoFromBytes`), and typed attribute-value extraction
(:cpp:func:`onnx::RetrieveValues`).
