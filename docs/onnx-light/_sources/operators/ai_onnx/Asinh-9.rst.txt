Asinh - version 9
=================

This page documents version **9** of operator **Asinh**. See :doc:`Asinh` for the latest version (since version 22).

- **Domain**: ``ai.onnx``
- **Since version**: 9

Computes the Asinh value of the input tensor element-wise.

**Inputs**

- **input** (*T*): Input tensor

**Outputs**

- **output** (*T*): The hyperbolic arcsine values of the input tensor computed element-wise

**Type Constraints**

- **T**: Constrain input and output types to float tensors.
  Allowed types: tensor(double), tensor(float), tensor(float16).
