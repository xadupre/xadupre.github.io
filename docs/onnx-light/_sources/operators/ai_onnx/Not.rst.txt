.. _op_ai_onnx_Not:

Not
===

- **Domain**: ``ai.onnx``
- **Since version**: 1

Returns the negation of the input tensor element-wise.

**Inputs**

- **X** (*T*): Input tensor

**Outputs**

- **Y** (*T*): Output tensor

**Type Constraints**

- **T**: Constrain input/output to boolean tensors.
  Allowed types: tensor(bool).
