Abs - version 1
===============

This page documents version **1** of operator **Abs**. See :doc:`Abs` for the latest version (since version 13).

- **Domain**: ``ai.onnx``
- **Since version**: 1

Absolute takes one input data (Tensor ) and produces one output data
(Tensor ) where absolute value, y = abs(x), is applied to
the tensor elementwise.

**Inputs**

- **X** (*T*): Input tensor

**Outputs**

- **Y** (*T*): Output tensor

**Type Constraints**

- **T**: Constrain input and output types to float tensors.
  Allowed types: tensor(double), tensor(float), tensor(float16).
