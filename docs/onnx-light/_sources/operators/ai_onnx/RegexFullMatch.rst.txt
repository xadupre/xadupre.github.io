RegexFullMatch
==============

- **Domain**: ``ai.onnx``
- **Since version**: 20

RegexFullMatch performs a full regex match on each element of the input tensor. If an element fully matches the regex pattern specified as an attribute, the corresponding element in the output is True and it is False otherwise. `RE2 <https://github.com/google/re2/wiki/Syntax>`_ regex syntax is used.

**Inputs**

- **X** (*T1*): Tensor with strings to match on.

**Outputs**

- **Y** (*T2*): Tensor of bools indicating if each input string fully matches the regex pattern specified.

**Type Constraints**

- **T1**: Inputs must be UTF-8 strings
  Allowed types: tensor(string).
- **T2**: Outputs are bools and are True where there is a full regex match and False otherwise.
  Allowed types: tensor(bool).
