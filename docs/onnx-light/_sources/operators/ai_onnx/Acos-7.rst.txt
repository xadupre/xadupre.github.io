Acos - version 7
================

This page documents version **7** of operator **Acos**. See :doc:`Acos` for the latest version (since version 22).

- **Domain**: ``ai.onnx``
- **Since version**: 7

Computes the Acos value of the input tensor element-wise.

**Inputs**

- **input** (*T*): Input tensor

**Outputs**

- **output** (*T*): The arccosine of the input tensor computed element-wise

**Type Constraints**

- **T**: Constrain input and output types to float tensors.
  Allowed types: tensor(double), tensor(float), tensor(float16).
