Atanh - version 9
=================

This page documents version **9** of operator **Atanh**. See :doc:`Atanh` for the latest version (since version 22).

- **Domain**: ``ai.onnx``
- **Since version**: 9

Computes the Atanh value of the input tensor element-wise.

**Inputs**

- **input** (*T*): Input tensor

**Outputs**

- **output** (*T*): The hyperbolic arctangent values of the input tensor computed element-wise

**Type Constraints**

- **T**: Constrain input and output types to float tensors.
  Allowed types: tensor(double), tensor(float), tensor(float16).
