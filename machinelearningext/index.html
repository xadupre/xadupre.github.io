This documentation goes through some elements useful to whoever wants to extend ML.net which is a open source machine learning library mostly written in C# and implemented by Microsoft. It contains documentation pages about the ML.net itself taken from the main repository or automatically built from it extended with custom code or experiments. These extensions are not yet available as a nuget package but the source can obtained by forking xadupre/machinelearningext.
