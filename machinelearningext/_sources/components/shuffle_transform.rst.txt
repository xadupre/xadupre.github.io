**Description**

Reorders rows in the dataset by pseudo-random shuffling.


**Parameters**

.. list-table::
    :widths: 5 5 5 20
    :header-rows: 1

    * - Name
      - Short name
      - Default
      - Description
    * - forceShuffle
      - force
      - False
      - If true, the transform will always provide a shuffled view.
    * - forceShuffleSeed
      - seed
      - 
      - The random seed to use for forced shuffling.
    * - forceShuffleSource
      - forceSource
      - 
      - If true, the transform will always shuffle the input. The default value is the same as forceShuffle.
    * - poolOnly
      - po
      - False
      - If true, the transform will not attempt to shuffle the input cursor but only shuffle based on the pool. This parameter has no effect if the input data was not itself shufflable.
    * - poolRows
      - rows
      - 1000
      - The pool will have this many rows
