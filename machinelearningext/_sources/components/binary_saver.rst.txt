**Description**

Writes data into a native binary IDV file.


**Parameters**

.. list-table::
    :widths: 5 5 5 20
    :header-rows: 1

    * - Name
      - Short name
      - Default
      - Description
    * - compression
      - comp
      - Deflate
      - The compression scheme to use for the blocks
    * - deterministicBlockOrder
      - det
      - False
      - If true, this forces a deterministic block order during writing
    * - maxBytesPerBlock
      - bpb
      - 83886080
      - The block-size heuristic will attempt to have about this many bytes across all columns per block, can be set to null to accept the inidcated max-rows-per-block as the number of rows per block
    * - maxRowsPerBlock
      - rpb
      - 8192
      - The block-size heuristic will choose no more than this many rows to have per block, can be set to null to indicate that there is no inherent limit
    * - silent
      - 
      - False
      - Suppress any info output (not warnings or errors)
