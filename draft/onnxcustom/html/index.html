onnxcustom: deploy, train machine learned modelsÂ¶

Examples, tutorial on how to convert machine learned models into ONNX, implement your own converter or runtime, or even train with ONNX, onnxruntime. The documentation introduces onnx, onnxruntime for inference and training. It implements training classes following scikit-learn based on onnxruntime-training enabling training linear models, neural networks on CPU or GPU. It implements tools to manipulate logs produced NVidia Profiler logs (convert_trace_to_json), tools to manipulate onnx graphs.

Section API summarizes APIs for onnx, onnxruntime, and this package. Section Tutorials explains the logic behind onnx, onnxruntime and this package. It guides the user through all the examples this documentation contains.
