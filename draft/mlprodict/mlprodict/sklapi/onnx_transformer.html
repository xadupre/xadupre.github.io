class mlprodict.sklapi.onnx_transformer.OnnxTransformer(onnx_bytes, output_name=None, enforce_float32=True, runtime='python', change_batch_size=None, reshape=False)Â¶

Bases: sklearn.base.BaseEstimator, sklearn.base.TransformerMixin, skl2onnx.algebra.onnx_operator_mixin.OnnxOperatorMixin

Calls onnxruntime or the runtime implemented in this package to transform input based on a ONNX graph. It follows scikit-learn API so that it can be included in a scikit-learn pipeline. See notebook Transfer Learning with ONNX for an example.

Parameters

  • onnx_bytes â€“ bytes

  • output_name â€“ string requested output name or None to request all and have method transform to store all of them in a dataframe

  • enforce_float32 â€“ boolean onnxruntime only supports float32, scikit-learn usually uses double floats, this parameter ensures that every array of double floats is converted into single floats

  • runtime â€“ string, defined the runtime to use as described in OnnxInference.

  • change_batch_size â€“ some models are converted for a specific batch size, this parameter changes it, None to avoid changing it, 0 to fix an undefined first dimension

  • reshape â€“ reshape the output to get a matrix and not a multidimensional array
