module ``mlprodict.asv_benchmark.template.skl_model_multi_classifier``

A template to benchmark a model
with :epkg:`asv`. The benchmark can be run through
file :epkg:`run_asv.sh` on Linux or :epkg:`run_asv.bat` on
Windows.
