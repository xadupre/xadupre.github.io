module ``onnx_tools.onnx_export``
=================================







Short summary
+++++++++++++

module ``mlprodict.onnx_tools.onnx_export``

Exports an ONNX graph in a way it can we created again
with a python script. It relies on :epkg:`jinja2` and :epkg:`autopep8`.
