You should look for existing issues or submit a new one. Sources are available on xadupre/scikit-onnxruntime. The following example is taken from Transfer learning with ONNX.

with open("rf_iris.onnx", "rb") as f:
    content = f.read()

ot = OnnxTransformer(content, output_name="output_probability")
ot.fit(X_train, y_train)

print(ot.transform(X_test[:5]))
