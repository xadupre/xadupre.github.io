Experimental ONNX repositories

mbext

modelbuilder + CI implementing discrepancies on small untrained models, a way to make copilot use CI and save a couple of manual steps

my-own-accelerator

a kind of automation of my own old habits

onnx

weekly statistics gathered from the latest onnx release on PyPI (wheel size, opset, operator/test counts)

onnx-light

onnx without protobuf, a better shape inference alogithm, a replacement implemented the same API but offering more freedom

yet-another-onnx-builder

a fusion of all existing converters for scikit-learn, xgboost, lightgbo, tensorflow, pytorch, scikit-survival, category_encodes, SQL, polars, pandas, ...

yet-another-onnxruntime-extensions

simplified project to implement onnxruntime custom kernels in CPU and CUDA
